# What does this mod do?
This mod replaces some monster enemy audio with AI-generated songs inspired by those enemies.<br>
It also adds these songs to the store and the museum boombox.

### client-side mod, safe for multiplayer
not all players need to have this mod installed

## audio replacament for monsters
headman - chase sound is replaced by "the malvado cabeza"
(more enemies may be added in the future)

## shop music and boombox (randomized)
- "el malvado cabeza" [headman]
- "el anciano" [hunter]
- "pato" [duck]
- "el oculto" [hidden]
